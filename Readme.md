# Logistic Regression From Scratch (No sklearn)

This project implements **Logistic Regression from scratch** using only **NumPy and Matplotlib**.
No machine learning libraries like `sklearn` are used.

The model predicts the **probability of heart disease** based on age using the **sigmoid function**, **log loss**, and **gradient descent**.
